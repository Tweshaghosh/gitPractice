 Having easy access to the entire history of a file can save the time to create the file again or to code the material again. It is important to make better improvements to your product rather than wasting time on the same thing over and over again.

 As a programmer I would want my code to be saved so if I want to use a part of a code or even just to create another project I can go back to it.

 Commit messages helps everyone else know that a change was made in the file. It helps with finishing assignments.
